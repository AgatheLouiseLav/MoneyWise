## Key Learnings

Our journey with MoneyWise has been an great learning experience, from technical to collaborative aspects. As we developed this full-stack CRUD web app, we gained valuable insights that have polished our skills and teamwork. Here are some key learnings from our MoneyWise project:

1. **Collaboration**: Building a complex project like MoneyWise required good collaboration between backend and frontend developers. Regular communication, code reviews, and addressing challenges as a team greatly contributed to our success.

2. **Graphs**: The integration of Charts.js present data dynamically with interactive graphs. This taught us how to visualise data and present them in an easy and clear format.

3. **Version Control**: Using Git and GitHub throughout the project underscored the importance of version control in collaborative development, enabling efficient code management and tracking changes.
